# LE-Synoptic-Stem-Flux-discrete-data
Code for processing discrete data from tree stem fluxes at the COMPASS Lake Erie sites 

Sites: CRC, PTR, OWC
Zones: Upland and Transition for each site
n = 8 trees in each zone
technique: pvc collars installed from 80 to 120 cm heights, Licor 7810, incubation time from 3 to 5 minutes 
Dead-band adjusted individually for each measurement
QAQC flags criteria based on the CO2 flux behave: 
    1-Observations with fCO2<0  
    2-Observations with initial CO2 concentration > 550ppm and r2<0.7 
    3-Visual checks for concentration drops (leak) during measurements
